# About

I am Shubham Patil, a Computer Science student focused on backend, systems programming, Linux tooling, local AI, and low-level fundamentals.

My strongest technical interests are:

- Rust, C, C++, and Go.
- Backend services and infrastructure.
- Operating-system concepts: processes, pipes, file descriptors, shells, IPC.
- Local-first AI systems using Ollama, embeddings, and retrieval.
- Linux desktop tooling and personal automation.

This garden is where I turn projects and learning into durable public notes. It is intentionally narrower than my private second brain.

## Current Direction

I want my work to converge around:

- building systems that actually run
- understanding fundamentals instead of only using frameworks
- writing clearly about implementation decisions
- shipping projects with demos, benchmarks, and honest status notes
