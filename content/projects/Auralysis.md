# Auralysis

## Status

Completed hackathon/supporting project.

## Summary

A deployed medical AI platform built in a 24-hour hackathon. It accepts MRI or X-ray images and produces disease predictions, structured reports, patient-friendly summaries, and voice output.

## Stack

- FastAPI
- TensorFlow / EfficientNetV2B0
- LLM API integration
- gTTS
- Supabase / PostgreSQL
- JWT authentication
- Docker
- Render
- Datadog
- Flutter and React clients

## Pipeline

```text
image upload
  -> disease prediction
  -> structured medical report
  -> patient summary
  -> voice output
  -> persisted history
```

## Why It Matters

It proves fast shipping, full-stack integration, applied ML, API orchestration, auth, deployment, and monitoring. It is less central to the backend/systems narrative than the Rust projects, but strong for hackathon and applied-AI contexts.
