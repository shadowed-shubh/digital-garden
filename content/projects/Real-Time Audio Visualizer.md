# Real-Time Audio Visualizer

## Status

Complete or mostly complete supporting project.

## Summary

A C++/Raylib music visualizer using Fourier Transform-based signal processing to render frequency information in real time.

## Technical Surface

- C++.
- Raylib rendering.
- Frequency-domain signal processing.
- FFT / frequency extraction.
- Frame-synchronized rendering loop.
- Real-time performance constraints.

## Why It Matters

This project shows a different side of systems work: performance-sensitive graphics and signal processing rather than backend APIs.
