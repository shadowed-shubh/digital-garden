# SignBridge

## Status

Supporting prototype.

## Summary

A FastAPI backend that receives sign-detection events over WebSockets and translates sign sequences into natural English using a local Qwen/Ollama flow.

## Stack

- Python
- FastAPI
- WebSockets
- Ollama
- Qwen

## What It Demonstrates

- Real-time backend communication.
- Rolling context/memory handling.
- Local LLM integration.
- Translation pipeline design.
