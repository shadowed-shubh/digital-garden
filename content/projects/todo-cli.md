# todo-cli

## Status

Small supporting project.

## Summary

A Rust terminal todo app using ncurses and plain-text storage.

## Features

- Vim-style `w`/`s` navigation.
- Enter to toggle completion.
- `q` to quit and save.
- Persistent plain-text storage.

## What It Demonstrates

- Rust file I/O.
- Terminal UI design.
- ncurses integration.
